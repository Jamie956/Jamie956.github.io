# Typora quickly start
## 快捷键

ctrl + 1 //标题1

ctrl + 2 //标题2

...

ctrl + 5 //标题5

ctrl + u //下划线

ctrl + b //加粗

ctrl + i //斜体

ctrl + k //超链接 [百度](http://www.baidu.com)

ctrl + t //表格

\> + 空格 //引用

ctrl + l //选中行

ctrl + d //选中词

alt + shift + 5 //删除线

ctrl + f //搜索

ctrl + h //搜索替换

ctrl + shift + i //插入图片

\- + 空格 //无序列表

1 + . //有序列表
