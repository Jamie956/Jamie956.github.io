SSH

  1. 生成SSH密钥
    ssh-keygen -t rsa -C "your@eamil.com"
  2. 查看
    cat ~/.ssh/id_rsa.pub
  3. 把SSH添加到git
  4. 测试
    ssh -T git@github.com
  5. 全局配置用户信息
    git config --global user.name "yournamee"
git config --global user.email "your@eamil.com"
