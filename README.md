## Gitbook 记笔记，美滋滋～

| 命令                       | 说明              |
| -------------------------- | ----------------- |
| npm i gitbook-cli       | 安装 gitbook  |
| touch README.md SUMMARY.md | 创建索引结构      |
| gitbook init               | 生成索引结构文件  |
| gitbook build              | 编译              |
| gitbook serve              | 编译并开启 server |
| npm i gh-pages | 安装 gh-pages |
| gh-pages -d _book | 将文件夹_book 的内容同步到分支 gh-pages |

### 快速开始

```js
npm i
npm run build
npm run serve
npm run sync
```
